# AI Resume Analyzer

AI Resume Analyzer built using Python and Streamlit.

Features:
- Upload Resume PDF
- Skill Extraction
- ATS Match Score
- Missing Skills Detection
- Resume Suggestions
- Analyze Resume button
- Reset button

Tech Stack:
- Python
- Streamlit
- pdfplumber
- scikit-learn

Run project:

streamlit run app.py
